# Markdown-editor-MAKUMAKU
マークダウンエディターのまくまくです。できるだけ軽く手軽に動作するよう、設計しました。

## 設計思想
- 誰でも使えるよう、機能は必要最低限に。
- 非常に軽いレスポンス。
- CDNを使用せず、継続的な運用を
- HTMLへの変換も楽々
